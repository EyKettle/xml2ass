# XML Converter for Bilibili

English | [中文](./README_CN.md)

Convert bilibili Danmaku XML file to ASS file
that could load in common video player.

## Support Feature

- 4 types of Danmaku
- Danmaku color
- Font size

### Support Danmaku Type

- Top Danmaku
- Bottom Danmaku
- Scrolling Danmaku
- Reverse Scrolling Danmaku
